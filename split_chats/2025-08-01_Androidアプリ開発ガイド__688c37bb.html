🔧 まず何が必要か？

1. パソコン

  • Windows、Mac、Linux どれでもOK
  • メモリは最低8GB（できれば16GB以上）

2. Android Studio（公式の開発ツール）

  • Googleが提供しているIDE（統合開発環境）
  • 公式サイトから無料でダウンロード

3. Java か Kotlin の知識

  • どちらもAndroid開発に使えます
  • 現在は Kotlinが推奨
  • もし初心者なら、Kotlinを少し学んでから始めるのがおすすめ

✅ Step 1: 必要なものを準備

✅ パソコンのスペック

  • Windows 10 or 11（64bit）
  • 空き容量：5GB以上
  • メモリ：8GB以上推奨

ツール名 説明
Flutter SDK Flutter本体
Android Studio エミュレータや開発環境として使用

✅ Step 2: Flutter SDKをインストール

  1. 以下のURLにアクセス：
    📎 Flutter公式ダウンロードページ
  2. 「Windows」タブを選択
  3. 「flutter_windows_x.x.x-stable.zip」をダウンロード
  4. ZIPファイルを任意の場所に展開（例：C:\src\flutter）

✅ Step 3: 環境変数を設定

  1. C:\src\flutter\bin のパスをコピー
  2. Winキー → 「環境変数」と検索 → 「システム環境変数の編集」
  3. 「環境変数」→ 「Path」→「編集」→「新規」で上記のパスを追加
  4. OKで閉じる

✅ Step 4: Flutterが使えるか確認

  1. Winキー + R → cmdでコマンドプロンプトを開く
  2. flutter doctorと入力
    → エラーが出なければ成功！（初回は少し時間がかかる）

✅ Step 5: Android Studioをインストール

  1. Android Studio公式からダウンロード

  2. インストール時、以下を有効にする：
    - Android SDK
    - Android SDK Command-line Tools
    - Android Emulator

  3. 起動後、「Plugins」から Flutter プラグインを検索してインストール（Kotlinも一緒に入ります）

✅ Step 6: Flutterプロジェクトを作成してみる

  1. Android Studio → 「New Flutter Project」
  2. 「Flutter SDK Path」には C:\src\flutter などを入力
  3. 「Flutter Application」を選択
  4. アプリ名・保存場所を入力
  5. 作成完了後、すぐにデモアプリが表示されます！

✅ Step 7: 実行してみよう！

  • エミュレータの起動：ツールバーからPixelなどを選んで起動
  • 「▶（再生）」ボタンを押してアプリを起動
  • 画面に「You have pushed the button…」が表示されれば成功！

CドライブとDドライブって何が違うだっけ

いい質問です。
「Cドライブ」と「Dドライブ」の違いは、基本的には「どこにあるストレージか（何用の領域か）」という物理的または論理的な区分けです。

🖥️ 【基本】CドライブとDドライブの違い

項目 Cドライブ Dドライブ
一般的な用途 Windows本体（OS）やアプリのインストール先 データ保存・バックアップ・アプリ用領域など
初期状態 常に存在 ないこともある（PC構成次第）
実体 たいてい内蔵SSD/HDDの第1パーティション 別のパーティション or 別ドライブ（物理的に別のSSD/HDD）
フォーマット NTFS（Windows標準） 同じくNTFSが多い

🧠 もう少し具体的に言うと：

✅ Cドライブとは？

  • Windowsの本体（OS）が入っているドライブ
  • ソフトの初期インストール先（Program Files など）
  • ユーザーの デスクトップ や ドキュメント も基本ここ
  • OSが最も頻繁にアクセスするため、SSD化されていると高速

✅ Dドライブとは？

  • 一般的にはデータ保存や別アプリ用
  • 以下のようなパターンが多い：
  • Cドライブと同じ物理ストレージ上の別パーティション
  • 別の物理HDDまたはSSD
  • リカバリー領域（まれに）
  • 外付けドライブ（USBなど）

💡 確認したいときは…

🔎 「エクスプローラー」→「このPC」から見ると：

  • 「C:」は OS、ユーザーデータ、プログラムなどがぎっしり入ってるはず
  • 「D:」は空っぽだったり、特定のアプリ専用だったりすることも

🛠️ 開発でどちらにインストールすべき？

目的 推奨ドライブ
FlutterやAndroid Studioなど開発環境 Cドライブ（SSD）が圧倒的に速い
プロジェクトファイルやバックアップ Dドライブでも可（特に容量重視のとき）

📌 注意点

  • Cドライブは容量が逼迫するとPCの動作が極端に遅くなる
  • Dドライブが物理的に別ドライブなら、読み書きを分散できて快適になることもある

✅ まとめ

Cドライブ Dドライブ
Windows本体・アプリ・システムが入る データ保存・開発プロジェクト用など自由度が高い
高速なSSDであるべき 容量が多ければプロジェクトにも最適
基本的に最も重要な領域 用途は人それぞれ・空き容量次第
